# jupyterlab-starters

Parameterized file and directory starters for JupyterLab.

# What's a starter?

- [x] A single file
- [x] A directory
- [ ] A [cookiecutter][]
- [ ] A python function

... that shows up in your JupyterLab at the click of a button

[cookiecutter]: https://github.com/cookiecutter/cookiecutter
